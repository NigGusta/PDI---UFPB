# PDI---UFPB
Sistema capaz de:  Carregar e exibir imagens RGB no ambiente Python.  Aplicar diferentes filtros para convolução e correlação em imagens.  Implementar um sistema de correlação com parâmetros ajustáveis, incluindo offset, passo e função de ativação ReLU.  Desenvolver uma versão 3D, considerando os três canais de cor simultaneamente.
